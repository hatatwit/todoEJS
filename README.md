# todoEJS

A simple to-do list lets users add or delete items stored on MongoDB.

## Installation

Install Node.js on Windows/Mac

```bash
$ git clone https://github.com/hatatwit/todoEJS.git
$ cd todoEJS
$ npm install
$ npm start

```
